:h2:Build from source code

	Git repository: <a href="https://github.com/performous/performous.git">https://github.com/performous/performous.git</a> (see <a href="https://github.com/performous/performous/wiki/Building-and-installing-from-source" target="_blank">instructions</a>)

If you are building Performous yourself or wish to do any development on it, you should always use the git master branch rather than releases. The master branch is relatively stable as all development is done in separate branches but it often contains bugfixes and improvements that haven't been released yet.
