Other Similar Games

Nothing quite like Performous but there are interesting projects here. Let us know if anything worth a mention is missing.

Singing games:

  • UltraStar Deluxe - An open-source clone of SingStar
  • KaraokeParty (Flash) - The most popular browser game
  • KaraokeGame (Flash) - Has very good pitch detection (similar to Performous)

Guitar/band games:

  • Frets on Fire - The original PC guitar game, written in Python and no longer developed
  • FoFiX - FoF on steroids, a fork that adds a ton of new features
  • Phase Shift (Windows only) - A new band game, using FoF file format

Dance games:

  • StepMania - The original open-source dance game
  • PyDance - A dance game written in Python
